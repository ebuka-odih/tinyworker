# Run and deploy your AI Studio app

This contains everything you need to run your app locally.

View your app in AI Studio: https://ai.studio/apps/7f5d2e6d-6946-41d9-b71f-0dc6a2ddae28

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `VITE_API_BASE_URL` in `.env.local` (or use the default `http://localhost:4000/api`)
3. Run the app:
   `npm run dev`
